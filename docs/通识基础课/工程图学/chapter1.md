!!! tip
    点的投影规律：a'a垂直X轴，a'a"垂直Z轴，**aa`x`=a"a`z`**(重要)

![alt text](image-8.png)

点的三面投影与直角坐标的关系：
- 点的投影反映了点的坐标值
- 点的一个投影反映了点的两个坐标，若已知点的两个投影，则点的xyz三个坐标就可以确定。
